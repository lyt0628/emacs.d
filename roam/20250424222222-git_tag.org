#+title: git/tag

|--------------------------------------------------------------+----------------------------------|
| Command                                                      | Note                             |
|--------------------------------------------------------------+----------------------------------|
| git tag                                                      | 列出所有标签                       |
|--------------------------------------------------------------+----------------------------------|
| git tag -a <tag_name> -m "comment"                           | 创建附注标签（拥有打标签元信息的标签） |
|--------------------------------------------------------------+----------------------------------|
| git show <tag_name>                                          | 查看 标签的 提交信息                |
|--------------------------------------------------------------+----------------------------------|
| git tag <tag_name>                                           | 创建轻量标签(对提交节点的引用)        |
|--------------------------------------------------------------+----------------------------------|
| git log --pretty=oneline && git tag -a <tag_name> <checksum> | 用历史提交的校验和来为历史提交打标签   |
|--------------------------------------------------------------+----------------------------------|
| git push --tags                                              | 把标签提交到远程仓库                |
|--------------------------------------------------------------+----------------------------------|
| git tag -d <tag_name>                                        | 删除标签                           |
|--------------------------------------------------------------+----------------------------------|
| git push --delete <tag_name>                                 | 删除远程仓库的标签                  |
|--------------------------------------------------------------+----------------------------------|
| git push :refs/tags/a_tag                                    | ： 划分为前后两部分，               |
|                                                              | 前面的部分会代替后面部分，           |
|                                                              | ：前面为空，表达删除                |
|--------------------------------------------------------------+----------------------------------|
